# M E R G E - JavaFX Puzzle Game

A classic puzzle game developed using Java and JavaFX.
![Uploading 11.png…]()

## Overview
A grid-based logic puzzle game where players merge dice to achieve the highest score. This project was created to practice GUI development, object-oriented programming (OOP), and event handling in Java.

## Features
- Interactive drag-and-drop gameplay.
- Logic-based merging rules.
- Real-time score tracking and high score system.
- Graphical user interface powered by JavaFX.

## Prerequisites
- **JDK**: 11 or higher.
- **JavaFX SDK**: Required for compilation (since Java 11).
- **IDE**: IntelliJ IDEA or Eclipse.

## How to Run
1. Clone this repository.
2. Ensure your IDE is configured with the **JavaFX module-path**.
3. Build and run the `Main.java` file.
